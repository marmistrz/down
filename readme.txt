○ はじめに

 Down!!は、針を避けながら下へ下へと下りていくという「だけ」の、
 シンプルなアクションゲームです。

○ 動作環境

 Windows95,98,Me,NT,2000,XPで動くと思われます。
 
 またlinuxやFreeBSD等のunix系OSでも動作します。詳しくは作者のサイト
  (http://mono.kmc.gr.jp/~yhara/)をご覧下さい。
 理論的にはMacOS Xでも動作するはずです…が、まだ簡単ではないようです。
  (参考：http://www2s.biglobe.ne.jp/~nunokawa/wiki.cgi?page=Ruby%2FSDL+on+Mac+OS+X)

○ インストール・アンインストール

 ・インストール：アーカイブ内の全てのファイルをハードディスク上に
   展開してください。
 ・アンインストール：レジストリ等は使用していませんので、ファイルを消す
   だけでOKです。

○ 遊び方

 カーソルキーの左右で左右に移動して、下へと下りていってください。
 針に刺さると画面右の体力ゲージが減っていきます。体力が０になると
 ゲームオーバーです。

 次のようなアイテムが出現することもあります。
 [種類]        [効果]
 ☆            一定時間無敵になり、針に刺さってもダメージを受けなくなる
 パラシュート  落下速度が遅くなる
 重り          落下速度が速くなる

 パラシュートをとった状態で針に刺さると、パラシュートは割れてしまいます。
 また重りはとると落下が速くなってしまうので、なるべく避けた方が良いです。
 …が、重りをとったまま無敵状態になると…？

 １段落ちるごとに得点が１点加算されます。別に速く降りたからと言って
 得点が加算されるわけではないので、気を付けてゆっくり降りていってください。

○ 謝辞

 Down!!はRuby/SDLを用いて製作されました。
  * オブジェクト指向スクリプト言語Ruby   (http://www.ruby-lang.org/ja/)
  * SDL(Simple DirectMedia Layer)              (http://www.libsdl.org/)
  * Ruby/SDL                  (http://www.kmc.gr.jp/~ohai/rubysdl.html)
  
 RubyスクリプトをWindowsの実行形式ファイル(.exe)に変換するのにExerbを
 使用しています。
  * Exerb Project                        (http://exerb.sourceforge.jp/)

 起動時の、Window/Fullscreenを選択するフォームに、vrubyおよび
 FormDesignerを使用しています。
  * VisualuRuby計画(仮称) (http://www.osk.3web.ne.jp/~nyasu/software/vrproject.html)
  * FormDesigner (雪見酒さんのページ)      (http://homepage3.nifty.com/Yukimi_Sake/)

 それぞれの作者の方々に感謝します。
